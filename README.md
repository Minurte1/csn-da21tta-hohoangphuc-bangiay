## Hướng dẫn cài đặt dự án
Hướng dẫn cài đặt dự án bào gồm máy chủ Node.js và React

## Yêu Cầu

- Dbeaver
- Nodejs Express
- npm
- Reactjs

## Cài Đặt

1. **Clone Repository:**
   ```bash
   git clone https://github.com/Minurte1/csn-da21tta-hohoangphuc-bangiay-Javascript


2. **Cài Đặt Dependencies:**

   - Chuột phải vào thư mục csn-da21tta-hohoangphuc-bangiay chọn Open in integrated Terminal
    ```bash
   npm install
   ```

4. **Cấu Hình Môi Trường:**
   
   - Tạo một tệp `.env` từ tệp `.env.example` và điền thông tin cần thiết (env có sẵn trong dự án).

6. **Chạy Ứng Dụng:**

   - Mở visual studio code Open dự án
   - Chuột phải vào tệp package.json chọn Open in integrated Terminal
   ```bash
   npm run start
   ```

   - Hoặc sử dụng `nodemon` để tự động khởi động lại ứng dụng khi có thay đổi (đã được cài sẵn):

   ```bash
   npm install -g nodemon
   nodemon
   ```
